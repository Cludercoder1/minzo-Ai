# MinzoAI Quick Start Guide

This file will help you get MinzoAI running quickly on your computer.

## 1. Prerequisites
- Node.js 18+ (https://nodejs.org/)
- npm (comes with Node.js)
- (Optional) Docker & Docker Compose

## 2. Backend Setup
1. Open PowerShell or Command Prompt.
2. Navigate to the backend folder:
   ```powershell
   cd backend
   ```
3. Install dependencies:
   ```powershell
   npm install
   ```
4. Start the backend server:
   ```powershell
   node server.js
   ```
   The backend runs at http://localhost:3001

## 3. Frontend Setup
1. Open a new terminal window.
2. Navigate to the client folder:
   ```powershell
   cd client
   ```
3. Install dependencies:
   ```powershell
   npm install
   ```
4. Start the frontend:
   ```powershell
   npm start
   ```
   The frontend runs at http://localhost:3000

## 4. Using Docker (Optional)
If you have Docker and Docker Compose installed, you can run both backend and frontend with:
```powershell
docker-compose up --build
```

## 5. Troubleshooting
- If you see errors about missing modules (like 'express'), run `npm install` in the correct folder.
- If you see errors about native modules (like 'better-sqlite3'), the backend will use file storage automatically.
- Always use PowerShell or Command Prompt, and avoid folders with special characters in the path if possible.
